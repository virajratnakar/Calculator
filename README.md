# Fancy PEMDAS Calculator

to access this website on any device, use the link given below:

 
A stylish, interactive web-based calculator that follows PEMDAS (Order of Operations) rules. Built using HTML, CSS, and JavaScript, this calculator combines functionality with a modern glassmorphism UI.

🚀 Features

🎨 Modern UI Design

Glassmorphism card layout

Smooth gradients and shadows

Responsive design for mobile & desktop

🔢 Interactive Number Input

Clickable number buttons

Keyboard support (0–9 + Enter)

➕ Operator Support

Addition (+)

Subtraction (−)

Multiplication (×)

Division (÷)

🧠 PEMDAS Logic

Automatically follows correct order of operations:

Parentheses (not implemented yet)

Exponents (not implemented yet)

Multiplication & Division

Addition & Subtraction

⚡ Animations

Button flash effects

Smooth transitions

🔄 Reset & Reuse

Clear button (C)

Continue calculations after result

📸 Preview

A sleek calculator interface with glowing buttons and dynamic interaction.

🛠️ Technologies Used

HTML5

Glassmorphism effects

Vanilla JavaScript

🎮 How to Use

Enter a number using:

On-screen buttons OR

Keyboard input

Click Enter

Choose an operator (+, −, ×, ÷)

Enter the next number

Press = to get the result

⌨️ Keyboard Support
Key	Action
0–9	Input numbers
Enter	Submit number
🔄 Reset Calculator

Click the C button to:

Clear all inputs

Restart calculation

⚠️ Limitations

No support for:

Parentheses ()

Exponents ^

Decimal validation edge cases

Basic error handling only
